Descriere a jocului:
Simulare Boids cu Pygame:
Aplicația simulează mișcarea și comportamentul boid-urilor într-un mediu grafic.
Fiecare boid are o poziție, o viteză, o accelerație, și anumite comportamente precum separare, aliniere și coeziune.
Interfață Grafică:

Se folosește Pygame pentru a crea o fereastră și a afișa simularea.
Utilizatorul poate interacționa cu parametrii și comportamentele boid-urilor prin intermediul unei interfețe grafice.
Panou de Control:

Există un panou de control care conține butoane pentru resetarea simulării și afișarea/ascunderea interfeței grafice.
Limbaje / Tehnologii Folosite:
Python cu Pygame:

Dezvoltarea aplicației s-a realizat în limbajul Python.
Pygame a fost utilizat pentru manipularea graficii și crearea interfeței.
Structură Modulară:

Codul este organizat în fișiere separate precum boid.py, tools.py, matrix.py, constants.py, și ui.py.
Instrucțiuni de Rulare / Folosire:
Instalare Dependințe:

Asigurați-vă că aveți Python și pygame instalate pe sistem.
Rulare Aplicație:

Deschideți terminalul și navigați la directorul aplicației.
Rulați scriptul principal folosind comanda python nume_script.py.
